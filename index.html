Alt-text generator bookmarklets

Alt-text for post images

This bookmarklet generates alt-text for any image block you select in a post. Unfortunately featured images aren't exposed in a post in the same way, so it only works for any image block.

Instructions

  • Mouse-down on the following link (but don't complete the click): Post image alt-text and drag it up to your toolbar
  • Navigate to a post page
  • Click on an image
  • Click on the bookmarklet. It will generate some alt-text and show it to you in a pop-up
  • Edit the resulting alt-text if you need, then click "I like it!"
  • Repeat on any other images on the page
  • Save your post and you're done!

Test vision language models (VLMs) against images in our library

This bookmarklet allows its users to to get recommendations of alt-text images from two (soon more) VLMs as a way to test those models. This is just a testing tool to see if automated alt-text is even an option. If it is we can more deeply integrate automated alt-text into our processes.

Instructions

  • Mouse-down on the following link (but don't complete the click): Alt-text generator and drag it up to your toolbar
  • Navigate to a WordPress image library:
    • CPR
    • Denverite
  • Make sure you are in Grid Mode:
  • Click an image in the library and it will open up the Attachment Details modal.
  • Click your bookmarklet and wait a few seconds
  • An alert will pop up showing two (soon more) alt-text attempts from two different VLMs

That's it so far. As you can see there's no automated way to add the alt-text to the image. That's just because this is only a proof of concept and isn't designed for production.

At least one more model coming soon!
