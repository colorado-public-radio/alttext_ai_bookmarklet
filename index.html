Alt-text for post images

This bookmarklet generates alt-text for any image block you select in a post. Unfortunately featured images aren't exposed in a post in the same way, so it only works for any image block.

Instructions

First, mouse-down on the link below (but don't complete the click) and drag it up to your toolbar:

Post Image Alt-Text Generator

It will create a bookmarklet called "Post Image Alt-Text Generator".

Next, depending on the image that needs alt-text, follow the instructions below.

Featured image

  • Log into the dashboard and select a post to edit
  • Click the featured image at the top of the right rail
  • Once the featured image loads from the library, click the bookmarklet. It will generate some alt-text and show it to you in a pop-up
  • Edit the resulting alt-text if needed, then click "I like it!"
  • Click "Set featured image"
  • Save your post and you're done!

Post images

  • Log into the dashboard and select a post to edit
  • Click on a post image
  • Click the bookmarklet. It will generate some alt-text and show it to you in a pop-up
  • Edit the resulting alt-text if needed, then click "I like it!"
  • Repeat on any other images on the page
  • Save your post and you're done!

[Old] Test Vision Language Models (VLMs)

This bookmarklet allows users to get recommendations of alt-text for images from our library using different VLMs. This is a testing tool to evaluate if automated alt-text is viable.

Instructions

  • Mouse-down on the following link (but don't complete the click): Alt-text Generator (Old) and drag it up to your toolbar
  • Navigate to a WordPress image library:
    • CPR
    • Denverite
  • Make sure you are in Grid Mode:
  • Click an image in the library to open the Attachment Details modal
  • Click your bookmarklet and wait a few seconds
  • An alert will pop up showing alt-text attempts from different VLMs

This is a proof of concept and doesn't automatically add alt-text to images.
